Glossary

  1. API (Application Programming Interface): A set of rules that allows two software applications to communicate with each other and exchange data.

  2. Body doubling effect: When a person is more motivated to be productive in the presence of others, rather than alone at home.

  3. Git: Version Control

  4. Integrated Development Environment(IDE): Software application that provides facilities for software development.

  5. User Interface (UI): Visual elements of a software product.

  6. User Experience (UX): Overall experience and interaction a user has with a product.
