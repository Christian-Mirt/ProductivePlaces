Justify current development tools: VS Code: This IDE allows easy integration with pushing to GitHub, along with an integrated terminal for installing dependencies. Version Control: We are using GitHub for version control as it is the industry standard for codebase collaboration. Continuous Integration and Deployment: Our version control software has an integrated continuous integration feature called GitHub Actions. We decided on his tool due to it being an easy extension to our existing version control workflow.
Justify current design: The design focuses on simplicity, accessibility, and user-friendliness. Features like advanced filtering address users' specific needs while the community reviews section builds trust and enhances decision-making. Additionally, a responsive layout ensures usability across various devices.
Justify architectural decisions: The group has only discussed the schema for the database in regards to architectural decisions. The attributes for each table is separated in a way that allows for data to be stored and retrieve in an efficient manner while still retaining the necessary information to display to the end user.
Justify selected frameworks: ReactJS: For building a interactive UI NodeJS: This backend was selected to complement javascript throughout the stack, and to build a scalable real-time application Express: For building apis and middleware Vite: this will help rapidly develop the application by using out of the box optimization of web patterns Google Maps: Reliable api for real time numbers of crowds and opening/closing times
Justify selected language(s): HTML will be used for the structure of our web application for the frontend. CSS will be used for the beautiful presentation of our frontend elements in the web application. MySQL will be used for the database design of our web application, which will connect with the backend. JavaScript will be used to make the frontend website elements more interactive, such as downloading files. Additionally, JavaScript will also be used for the backend, which will connect with the frontend via a server.
